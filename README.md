# 🔥 Визуализация данных о пожарах на карте мира 🔥

🌍 Интерактивное приложение для визуализации данных о пожарах по всему миру с использованием данных CSV и библиотеки Plotly. 🗺️

## 📋 Описание

Приложение считывает данные о пожарах из CSV‑файла, извлекает географические координаты и уровень интенсивности пожаров, а затем отображает их на интерактивной карте мира. 🔥🗺️ Размер и цвет маркеров на карте соответствуют интенсивности пожара. 🔤

## ✨ Функциональность

* 💾 Чтение данных из CSV‑файла формата `MODIS_C6_1_Global_24h.csv`.
* 🔎 Парсинг географических координат (`latitude`, `longitude`) и показателя яркости (`brightness`).
* ⚠️ Обработка ошибок при чтении некорректных данных (пропуск строк с отсутствующими или невалидными значениями).
* 🎨 Визуализация данных на интерактивной географической карте с помощью Plotly Express.
* 🌠 Отображение интенсивности пожаров через размер и цвет маркеров.

## 🛠️ Требования

Для запуска приложения вам потребуется установить следующие библиотеки Python: 🐍

* `plotly` 📊 (для визуализации);
* `pathlib` 🗂️ (встроенная библиотека, не требует установки);
* `csv` 🗃️ (встроенная библиотека, не требует установки).

### 💿 Установка зависимостей

Вы можете установить необходимые библиотеки с помощью `pip`:

```bash
pip install plotly
```

## 🗂️ Структура проекта

В проекте всего несколько ключевых элементов:

* `README.md` 📄 — этот файл с инструкциями по установке и запуску.
* `fires_1.py` 🧪 — проверяет доступность файла и показывает структуру данных.
* `fires_2.py` 🔎 — выводит индексы и названия столбцов (широта — 0, долгота — 1, яркость — 2).
* `fires_3.py` 🚀 — извлекает данные и строит первую интерактивную карту с помощью Plotly (прототип приложения).
* `fires_4.py` 💻 — основной скрипт приложения.
* `fire_data/` 🗄️ — папка для хранения входных данных.
* `fire_data/MODIS_C6_1_Global_24h.csv` 📊 — CSV‑файл с данными о пожарах.

## ▶️ Запуск приложения

1. ✅ Убедитесь, что у вас установлен Python.
2. 💿 Установите зависимости, как описано в разделе «Установка зависимостей».
3. 📂 Поместите CSV‑файл с данными о пожарах в папку `fire_data` и назовите его `MODIS_C6_1_Global_24h.csv`.
4. 🚀 Запустите скрипт:

```bash
python fires_4.py
```

5. 🖥️ После выполнения скрипта в вашем браузере откроется интерактивная карта с отмеченными пожарами.
